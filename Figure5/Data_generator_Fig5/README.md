# Figure 5 data generation

Simulations supporting Figure 5's multi-patch comparisons.

## Key scripts
- `main_YLE_multi_patches.R` – YLE simulations across connected patches.
- `main_Homing_many_patch_ideal_para.R` and `main_Homing_many_patch_non_ideal_para.R` – Homing drive scenarios.
- `main_Xshredder_many_patch_ideal_para.R` and `main_Xshredder_many_patch_nonideal_para.R` – X-shredder scenarios.
- `cube_XShredderY.R`, `generate_YLE_inheritance_cube.R`, `generate_Homing_inheritance_cube.R` – Build inheritance structures for each drive type.

Run the desired scripts in R from this directory. Outputs are saved locally for plotting.
