# Supplementary Figure C: Target viability sweeps

Simulations and plots for viability-targeting YLE scenarios.

## Contents
- `main_YLE_vary_rel_fy_target_viability_haploinsufficient.R` – Executes parameter sweeps for viability-targeting releases.
- `generate_YLE_inheritance_cube.R` – Builds the inheritance cube.
- `Process_large_data_create_summary_csv_files.R` – Summarises simulation outputs.
- `FigC.R` – Generates the supplementary figure (output saved as `FigC.png`).
- `mgdriveYLE_sweep_10yr_release_target_viability/` – Stored simulation outputs.

Run the main script to regenerate simulations, process with the summary script, then execute `FigC.R` to recreate the figure.
