Contains additional simulations and figures that accompany the main text. Each subfolder corresponds to a supplementary figure.

## Subfolders
- `Figure A` – Latin hypercube sensitivity analysis and BRT analysis for all parameters including varying release values.
- `Figure B` – Sensitivity analysis with fixed release size assumptions.
- `Figure C` – Target viability sweeps with haploinsufficient effects.
- `Figure D` – Target fertility sweeps over 5-year release windows.
- `Figure E` – Suppression efficacy and population peaks as a function of release size for fsRRDL.
- `Figure F` – Suppression efficacy and population peaks as a function of release size for YLE.
- `Figure G` – Transient population overshoot during the initial phase of the YLE suppression campaign.
- `Figure H` – Dynamics of male YLE carrier frequency with time for different release sizes when target gene is haplosufficient.
- `Figure I` – Suppression efficacy and population peaks as a function of release size comparing SMR, fsRRDL and YLE.
- `Figure J` – Functional resistance YLE simulations and plotted outputs.
- `Figure K` – Functional resistance heatmap summaries and comparison outputs.
- `Figure L` – Baseline and boosted YLE comparisons using population extrema summaries.
- `Figure X Extra Analysis` – Carrying capacity sensitivity analysis for fertility-targeting YLE scenarios.
- `Calculation of Rm generation Time` – Basic reproductive number and generation-time calculations.

Follow the subfolder READMEs for details on running simulations and producing figures.
