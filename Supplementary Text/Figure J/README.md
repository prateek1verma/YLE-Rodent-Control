# Supplementary Figure J: Functional resistance YLE

Simulations and plots for YLE scenarios with functional resistance assumptions.

## Contents
- `main_YLE_HIS_target_funR.R` - Runs the functional-resistance YLE simulations.
- `generate_YLE_inheritance_cube_funR.R` - Builds the functional-resistance YLE inheritance cube.
- `Fig_J.R` - Generates the supplementary figure.
- `Fig_J.png` - Rendered supplementary figure.

Run the main script to regenerate simulations for YLE with functional resistance while utilising the data/code from Fig 5a for all other strategies. Execute `Fig_J.R` to rebuild the figure while having data containing folders for each strategies in the same folder as `Fig_J.R`.
