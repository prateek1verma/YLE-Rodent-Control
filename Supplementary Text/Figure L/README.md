# Supplementary Figure L: Boosted YLE comparisons

Simulations and plots comparing YLE and boosted YLE strategies using maximum total population and minimum female population summaries.

## Contents
- `main_YLE_vary_rel_fy_target_fertility_haploinsufficient.R` - Runs the baseline YLE release-size sweep.
- `main_contritutive_YLE_vary_rel_xshred_target_fertility_haploinsufficient.R` - Runs the boosted YLE release-size and X-shredder sweep.
- `generate_YLE_inheritance_cube.R` - Builds the baseline YLE inheritance cube.
- `generate_boosted_YLE_constitutive_inheritance_cube.R` - Builds the boosted YLE inheritance cube.
- `Fig_L.R` - Generates the supplementary figure and summary outputs.
- `combined_peak_total_min_female_all_strategies.csv` - Summary table used for plotting.
- `Fig_L.pdf` - Rendered supplementary figure.
- `Fig_L.pptx` - Editable supplementary figure panel.

Run the simulation scripts to regenerate outputs, then execute `Fig_L.R` to rebuild the summary table and figure while having the data in the same folder as this R script.
